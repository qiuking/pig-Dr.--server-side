4 Update boar metadata
    url: http:ip:port/boars/${boar_id}    function: PUT 
    备注：${boar_id}变量代指boar db record的id属性，主键
    Header: Content-Type  application/json
    body:   {"UpdateBoar": {"key1": value1, "key2": value2}}
    body example:
{
    "UpdateBoar":{
        "ear_tag":3,
        "breed_acceptability":0.5
    }
}
    response example:
{
    "boar":{
        "entryday":null,
        "ear_lack":65,
        "breed_num":3,
        "birthday":null,
        "breed_acceptability":0.5,
        "ear_tag":3,
        "dormitory_id":3,
        "source_id":3,
        "category_id":3,
        "id":3
    }
}

5 Delete baor record

    备注：${boar_id}变量代指boar db record的id属性，主键
    Header: Content-Type  application/json
